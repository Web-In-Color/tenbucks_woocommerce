tenbucks.© est une incroyable boite à outils, simple, efficace, elle regroupe des dizaines de fonctionnalités essentielles pour votre site e-commerce.

== Description ==
This plugin **require WooCommerce** >= 2.3
Use Tenbucks addons to get more of your WooCommerce shop:

* Generate barcodes,
* Import products,
* Use Expeditor Inet (France),
* Send automaticly orders to your dropshipping suppliers

You can access to addons via an iframe in your back-office, or directly on our website (link on bottom of your back-office).


== Installation ==

1. Upload `tenbucks.zip` to the `/wp-content/plugins/` directory,
1. Activate the plugin through the 'Plugins' menu in WordPress,
1. Make sure you have a valid API Key and secret with read/write permissions. [See how](http://docs.woothemes.com/document/woocommerce-rest-api/ "Documentation")
1. Use *Tenbucks* tab to access addons via an iframe, or directly on our [website](https://apps.tenbucks.io/my_account/ "My account").
